Device and Project Information

  For this project, an Argon Wi-Fi Development Board is used in combination with a AITrip Heart Rate Sensor in order to measure and record information about a user's health. The recorded information is sent to the website to be stored and displayed back to the user. The website was created using HTML, CSS, and Javascript, while the hardware requires C++ code to function. The user is able to sign up with a device ID, login, and receive a detailed view of recent heart rate measurements with a graph. A physican is able to change the frequency of their patient's heart rate testing.
